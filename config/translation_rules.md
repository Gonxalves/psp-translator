# PSP Translation Rules

These rules MUST be applied to all French-to-English translations for Personnel Support Programs (PSP) documentation.

## Rule 1: Glossary Priority
- **Use first and foremost the translations contained in the glossary "Traduction - Lexique Solange"**
- Do NOT add capital letters if there are none in the glossary

## Rule 2: Formatting Preservation

### a. Bold Text
- Reproduce the bold formatting used in the text to be translated
- Preserve **bold** exactly as it appears

### b. Italic Text
- Reproduce the italics used in the text to be translated
- Preserve *italics* exactly as they appear

### b2. Underline Text
- Reproduce the underline formatting used in the text to be translated
- Preserve ++underline++ exactly as it appears

### b3. Strikethrough Text
- Reproduce the strikethrough formatting used in the text to be translated
- Preserve ~~strikethrough~~ exactly as it appears

### b4. Highlighted Text
- Reproduce the highlighted text used in the text to be translated
- Preserve ==highlight== exactly as it appears (yellow highlight)
- For colored highlights, use ==#COLOR:text== format

### b5. Colored Text
- Reproduce the colored text used in the text to be translated
- Preserve ::#COLOR:text:: format for colored text

### c. Font and Spacing
- Provide the text in a version ready to paste in **Times New Roman 12** with the exact spacing
- Maintain consistent font throughout

### d. Paragraph Spacing
- Reproduce the same spacing between paragraphs as in the text to be translated
- Preserve all line breaks and empty lines

### e. Title Capitalization
- In titles, capitalize only the first word, unless there are proper nouns
- Example: "Annual report for personnel services" (not "Annual Report for Personnel Services")

### f. Subheading Capitalization
- In subheadings, capitalize only the first word, unless there are proper nouns
- Same capitalization rule as titles

### g. Quotation Formatting
- Do NOT put quotations in italics
- Keep quotations in regular text

### h. Colon Spacing
- Do NOT put a space before the colon (:)
- English convention: "Example: text" (not "Example : text")

### i. Currency Symbol Placement
- The dollar symbol ($) must be placed BEFORE the number
- Example: $100 (not 100$)

## Rule 3: Special Terms - Do Not Translate

The following terms MUST remain in French and NOT be translated:

### a. City Names
- Do NOT remove hyphens in proper names of cities
- Example: Saint-Jean-sur-Richelieu (keep as-is)

### b. Specific Terms
- "bouton rouge" → Keep as "bouton rouge"
- "Encadré" → Keep as "Encadré"

### c. Regiment Names
- "6e Bataillon, Royal 22e Régiment" → Keep as-is
- "4e Bataillon, Royal 22e Régiment" → Keep as-is
- "Régiment de Saint-Hyacinthe" → Keep as-is

### d. Image Files
- Do NOT translate photo file names in .jpg format
- Example: "photo_ceremonie.jpg" → Keep as "photo_ceremonie.jpg"

## Rule 4: Specific Spelling Requirements

Use Canadian/British English spellings for the following terms:

| French Term | English Translation | Notes |
|------------|---------------------|-------|
| kilomètre | kilometre | NOT "kilometer" (Canadian spelling) |
| couleur | colour | NOT "color" (Canadian spelling) |
| rigueur | rigour | NOT "rigor" (Canadian spelling) |
| tuque | tuque | Keep French term (uniquely Canadian) |

## Rule 5: Technical Accuracy

- Maintain technical accuracy for military/PSP terminology
- Use proper military ranks and titles
- Ensure context-appropriate translations for administrative and operational terms
- When in doubt about technical terms, check TERMIUM Plus or OQLF

✓ Bold (**text**) preserved

✓ Highlight (==text==) preserved
✓ Colored text (::#COLOR:text::) preserved
✓ Times New Roman 12pt formatting ready
✓ Paragraph spacing matches original

✓ No italics on quotations
✓ No space before colons

✓ "bouton rouge", "Encadré", regiment names NOT translated
✓ Photo filenames NOT translated
